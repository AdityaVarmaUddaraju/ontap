= Set quotas to limit resource use
:toc: macro
:toclevels: 1
:hardbreaks:
:nofooter:
:icons: font
:linkattrs:
:imagesdir: ./media/

[.lead]
Add quotas to limit the amount of disk space the quota target can use.

You can set a hard limit, and a soft limit for a quota.

Steps
1. Click *Storage > Quotas*.
2. Click *Add*.
